# Chaos Project: SINDy Comparison

## Overview
Investigation of sparse regression methods (STLSQ vs SR3) within the SINDy framework applied to the Lorenz system.

## Methods
- Lorenz system simulation
- SINDy implementation (PySINDy)
- STLSQ and SR3 optimisation
- Noise robustness analysis
- One-step RMSE evaluation

## Results
Includes phase-space plots, coefficient recovery, and noise comparison studies.
